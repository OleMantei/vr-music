Classic Jukebox
A detailed classic Jukebox model in 2 styles:
clean and dirty.

LODs are included and a second UV map is added for lightmapping.
Main textures is 2048x2048 in size,
Glass texture 512x512.

Tri count
LOD0: 4747 tris
LOD1: 3229 tris
LOD2: 1818 tris
